# SR-011

# aiTrade 시그널 연구실

## 시장과 종목의 보이지 않는 손

### D+0 / D+1 / D+5 최종 결론 v1.0

---

## 연구 목적

본 문서는

* 시장 전체 (코스피/코스닥)
* 대표주 12종목

의 수급 데이터를 분석하여

D+0 (당일 단타)

D+1 (종가 베팅)

D+5 (스윙)

각 기간별

시장을 움직이는 주체와

상승/하락의 공통분모를 찾는 것을 목적으로 한다.

---

# D+0 당일

| 구분        | 시장 전체           | 종목                 |
| --------- | --------------- | ------------------ |
| 가장 중요한 주체 | 외현 + 기관 + 프로그램  | 외인 + 기관 + 프로그램     |
| 핵심 키워드    | 현물 자금 유입        | 강한 손들의 동조화         |
| 상승 공통분모   | 외현↑ 기관↑ 프로그램↑   | 외인↑ 기관↑ 프로그램↑ 개인↓  |
| 하락 공통분모   | 외현↓ 기관↓ 프로그램↓   | 외인↓ 기관↓ 프로그램↓ 개인↑  |
| 개인 영향     | 작음              | 역방향 성향             |
| 최종 결론     | 현물 자금이 시장을 움직인다 | 강한 손이 같은 방향이면 상승한다 |

### D+0 상승 SS

외인↑ + 기관↑ + 프로그램↑ + 개인↓

### D+0 하락 SS

외인↓ + 기관↓ + 프로그램↓ + 개인↑

---

# D+1 종가베팅

| 구분        | 시장 전체         | 종목                    |
| --------- | ------------- | --------------------- |
| 가장 중요한 주체 | 외선 + 프로그램     | 기관 + 프로그램             |
| 핵심 키워드    | 미래 방향 선행      | 기관 지속성                |
| 상승 공통분모   | 외선↑ 프로그램↑     | 기관↑ 프로그램↑             |
| 하락 공통분모   | 외선↓ 프로그램↓     | 기관↓ 프로그램↓             |
| 개인 영향     | 거의 없음         | 거의 없음                 |
| 최종 결론     | 외선이 미래를 먼저 안다 | 기관과 프로그램이 다음날 방향을 만든다 |

### D+1 상승 SS

시장 : 외선↑ + 프로그램↑

종목 : 기관↑ + 프로그램↑ + 개인중립

### D+1 하락 SS

시장 : 외선↓ + 프로그램↓

종목 : 기관↓ + 프로그램↓

# D+5 스윙

| 구분        | 시장 전체          | 종목              |
| --------- | -------------- | --------------- |
| 가장 중요한 주체 | 외선 + 외현 + 프로그램 | 기관 + 외인         |
| 핵심 키워드    | 외국인 선도력        | 기관 지속 매집        |
| 상승 공통분모   | 외선↑ 외현↑ 프로그램↑  | 기관↑ 외인↑         |
| 하락 공통분모   | 외선↓ 외현↓ 프로그램↓  | 기관↓ 외인↓         |
| 개인 영향     | 없음             | 미미              |
| 최종 결론     | 외선이 추세를 만든다    | 기관과 외인이 추세를 만든다 |

### D+5 상승 SS

시장 : 외선↑ + 외현↑ + 프로그램↑

종목 : 기관↑ + 외인↑ + 프로그램↑

### D+5 하락 SS

시장 : 외선↓ + 외현↓ + 프로그램↓

종목 : 기관↓ + 외인↓ + 프로그램↓

| 등급 | 시장             | 종목               |

| S  | 핵심 주체 2개 동조    | 외인+기관 또는 외인+프로그램 |

| C  | 강한 손 역방향       | 관망               |
| D  | 핵심 주체 모두 약세    | 하락 위험            |

## 핵심 한 문장

시장은 외국인 선물이 미래를 먼저 알고 움직이며,

종목은 외인·기관·프로그램이라는 강한 손들이 같은 방향으로 움직일 때 가장 크게 상승한다.

강한 손들의 동조화(Consensus)가 aiTrade의 최상위 시그널이 될 가능성이 높다.
