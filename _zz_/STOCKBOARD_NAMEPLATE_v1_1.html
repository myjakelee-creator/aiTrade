STOCKBOARD_NAMEPLATE_v1.1

목적: 화면에 표시하는 이름과 데이터 연결에 사용하는 내부 키를 분리한다.

핵심 원칙: UI 문구가 변경되더라도 데이터 연결은 내부 키를 기준으로 유지한다.
