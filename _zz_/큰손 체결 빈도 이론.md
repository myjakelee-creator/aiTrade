# SR-012

# Big Hand Frequency Theory (BHFT)

## 큰손 체결 빈도 이론 v1.0

---

## 연구 목적

큰손은 한 번의 큰 주문으로 시장을 움직이는 것이 아니라, 큰 체결을 여러 번 반복하면서 시장에 흔적을 남긴다.

BHFT는 **5천만원 이상 체결이 얼마나 자주, 얼마나 짧은 시간 안에, 어느 방향으로 반복되는지**를 측정하여 주가 급등과 급락의 초입을 찾는 이론이다.

---

# 1. 기본 가설

기존 가설 : 큰손은 1억, 5억, 10억을 한 번에 주문한다.

BHFT 가설 : 큰손은 5천만원 체결을 수십~수백 번 반복한다.

예시 :

* 소형주 : 5천만원 × 10건 = 5억
* 삼성전자 : 5천만원 × 100건 = 50억

총액은 다르지만 공통점은 **짧은 시간 안에 5천만원 이상 체결이 계속 반복된다**는 점이다.

---

# 2. 실제 기관 알고리즘

| 알고리즘    | 특징           | 체결 특징          |
| ------- | ------------ | -------------- |
| TWAP    | 일정 시간 균등 분할  | 일정 금액 반복       |
| VWAP    | 거래량 비례 분할    | 거래량 증가 시 체결 증가 |
| POV     | 시장 거래량 비율 유지 | 거래량 따라 체결 증가   |
| Iceberg | 숨겨진 주문       | 동일 체결 반복       |
| Sniper  | 특정 조건 순간 체결  | 짧은 시간 폭발       |

### 결론

큰손은 **큰 주문 한방**보다 **작은 체결의 반복** 형태로 시장에 흔적을 남긴다.

---

# 3. 큰손 기준 금액

### 1차 가설

큰손 체결 기준 = **5천만원 이상**

---

선정 이유

|   기준 | 장점        | 단점        |
| ---: | --------- | --------- |
| 3천만원 | 소형주 포착 우수 | 노이즈 증가    |
| 5천만원 | 균형 우수     | 특별한 단점 없음 |
|  1억원 | 대형주 적합    | 소형주 놓침    |

---

### 결론

**5천만원 고정 → Replay 검증 → 필요 시 변경**

---

# 4. 중요한 것은 금액보다 빈도

예시 :

* 소형주 : 5천만원 × 10건 = 5억
* 삼성전자 : 5천만원 × 100건 = 50억

총액은 다르다.

하지만 짧은 시간 안에 5천만원 이상 체결이 지속적으로 반복된다는 점은 동일하다.

### BHFT 핵심

큰손 판단 기준은 **금액보다 빈도(Frequency)** 가 더 중요하다.

---

# 5. 건수 기준은 무엇으로 보정할 것인가?

| 기준          | 장점        | 단점           |
| ----------- | --------- | ------------ |
| 시가총액        | 기업 체급 반영  | 유동성 반영 약함    |
| 전일 거래대금     | 실제 유동성 반영 | 전일 이벤트 왜곡 가능 |
| 당일 거래대금     | 실시간 반영    | 장초반 불안정      |
| 전일+당일 평균    | 균형        | 약간 복잡        |
| 20일 평균 거래대금 | 가장 안정     | 데이터 확보 필요    |

---

### BHFT 결론

1차 기준 = **전일 거래대금**

보조 기준 = **시가총액**

20일 평균 거래대금 = **Replay 검증용**

---

# 6. 중요 시간 구간

| 구간     | 의미    |
| ------ | ----- |
| 최근 30초 | 순간 폭발 |
| 최근 1분  | 단타 진입 |
| 최근 5분  | 추세 시작 |
| 당일 누적  | 세력 활동 |

### 가장 중요한 구간

**최근 1분 + 최근 5분**

이 두 구간의 5천만원 이상 체결건수와 증가속도를 가장 중요하게 본다.

# 핵심 한 문장

**큰손은 한 번에 시장을 움직이지 않는다.**

**5천만원 이상의 체결을 짧은 시간 안에 반복하며, 그 반복의 리듬(Frequency)이 주가 급등과 급락의 시작을 알려준다.**
