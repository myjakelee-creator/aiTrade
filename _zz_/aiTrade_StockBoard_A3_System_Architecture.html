aiTrade StockBoard A3 통합 시스템 구성도

REST 장마감 조회와 OpenAPI+ 실시간 스트리밍을 분리하고, Store → Signal → Ranking → Strategy → API → HTML 흐름으로 정리한 설계 문서

한 줄 요약

키움 REST는 장마감/조회 데이터를 가져오고, 키움 OpenAPI+ COM은 장중 실시간 데이터를 받습니다. 두 데이터는 StockBoard Store에 모이고, Signal Engine → Ranking Engine → Strategy Engine을 거쳐 후보 5종목과 TOP100 화면에 표시됩니다.
REST = 장마감 조회
OpenAPI = 실시간 수신
Store = 데이터 창고
Engine = 두뇌
HTML = 표시 전용
